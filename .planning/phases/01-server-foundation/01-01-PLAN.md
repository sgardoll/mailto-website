  <name>Task 1: Create setup package and server.py with port probe, pre-flight check, and browser open</name>
  <files>setup/__init__.py, setup/server.py</files>
  <behavior>
    - find_free_port(preferred=7331): if socket.bind(('127.0.0.1', 7331)) succeeds → return 7331; if OSError → bind to ('', 0) and return the assigned port. Never returns 5000 or 8080.
    - wait_for_port(port, timeout=5.0): loop socket.create_connection(('127.0.0.1', port), timeout=0.1) every 50ms until success or timeout; raises RuntimeError on timeout.
    - check_write_permission(project_root): calls os.access(project_root, os.W_OK); returns True/False. Does NOT attempt to create files.
    - open_browser_after_ready(url, port): starts a daemon threading.Thread that calls wait_for_port(port) then dispatches open: sys.platform == 'darwin' → subprocess.Popen(['open', url]); other platforms → webbrowser.open(url).
    - main(): (1) resolve REPO_ROOT = Path(__file__).resolve().parent.parent, (2) call check_write_permission(REPO_ROOT) — if False: print error to stderr and sys.exit(1), (3) call find_free_port(), (4) call open_browser_after_ready() in daemon thread, (5) call app.run(host='127.0.0.1', port=port, debug=False, use_reloader=False).
    - Flask GET / renders templates/index.html with context {'port': port}.
    - if __name__ == '__main__': call main().
    - Module-level: __main__.py is NOT needed; entry point is python -m setup.server (the if __name__ == '__main__' block handles this).
  </behavior>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| localhost → Flask | All requests come from 127.0.0.1 only; no network exposure |
| filesystem → server | Server reads REPO_ROOT for permission check; path is computed from __file__, not user input |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | find_free_port() | accept | Port is bound to 127.0.0.1 only; no remote attacker can influence port selection |
| T-01-02 | Denial of Service | Port exhaustion | mitigate | Fallback to socket.bind(('', 0)) ensures a port is always found even under pressure |
| T-01-03 | Elevation of Privilege | REPO_ROOT path traversal | mitigate | REPO_ROOT = Path(__file__).resolve().parent.parent is computed from the installed file path, not user input; os.access() check before any write |
| T-01-04 | Information Disclosure | Server starts on non-localhost | mitigate | app.run(host='127.0.0.1') — never '0.0.0.0'; only loopback is exposed |
</threat_model>
